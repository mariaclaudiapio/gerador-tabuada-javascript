<h1 align="center"> Gerador de Tabuadas escrito em JavaScript </h1>

O aplicativo gera uma tabuada de multiplicação do número escolhido no campo "Tabuada do". Para escolher até que número deve ser multiplicado, preencher o campo "Multiplicar até".
